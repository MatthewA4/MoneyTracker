                           QUICK START
================================================================================

Build:
  cd MoneyTracker/build
  cmake -G Ninja ..
  ninja

Test Basic:
  ./moneytracker -h

Test with Sample Data:
  ./moneytracker -i data/sample_bank.csv -a "Checking" -v

Test Multiple Files:
  ./moneytracker \
    -i data/sample_bank.csv -a "Bank" \
    -i data/sample_savings.csv -a "Savings" \
    -v

Console Only (No Excel):
  ./moneytracker -i data.csv --no-spreadsheet -v

Custom Categories:
  ./moneytracker -i data.csv --category-config my_categories.json -v

1. Multi-Format Date Parsing
   - Handles US, European, and ISO date formats
